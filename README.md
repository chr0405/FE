## 💚Friend란?

안녕하세요! Friend를 소개합니다!\
Friend를 한 마디로 소개하자면 "지인 매칭 100% 방지와 개인 정보 노출 부담 없는 교내 매칭 서비스"입니다\
100% 실제 사람이 매칭해주어 당신의 특별한 Friend를 찾도록 도와주는 프리미엄 교내 소개팅 매칭 서비스입니다
